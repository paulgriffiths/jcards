jcards
======

What is it?
-----------

**jcards** is a Java playing cards library. Includes classes and utilities
for representing playing cards, hands, and decks, including poker hand
evaluation.

Who maintains it?
-----------------
**jcards** is written and maintained by Paul Griffiths.
